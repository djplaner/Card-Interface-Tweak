    How to create cards

    Introduction¶

    There is a simple three step process to create cards and the card interface. The following sections work through each of these. Steps 1 and 2 only need to be done once for each Blackboard page on which you want to use the Card Interface. Step 3 is repeated for as many times as you want cards.

    Pre-requisites¶

    The following assumes that you have a Blackboard site for which you have permission to create content. It also assumes that within that course you have a content area or content folder on which you want to use the card interface.

    Step 1. Add the tweak javascript¶

    The Card Interface is implement with some Javascript that transforms the original Blackboard content into the Card Interface. For this to work, the Javascript must be added to the page using the following steps.

    1. Copy all of the tweak code on this page into your clipboard.
    2. Create a new, empty content item on your designated Blackboard content area/folder (see these instructions for help).
      It doesn’t matter what name you call this content item. It will not be visible to students.
    3. When editing the new content item paste the tweak code into the item using the HTML Code View button.
    4. Save and submit your changes.

    This step only needs to be done once.

    Step 2. Create a content item for the cards to appear¶

    The Javascript you added in the last step will generate the card interface, but it needs somewhere to put that interface. That’s the aim of this step.

    1. Create a new Blackboard content item on your page.
    2. Give it the name Card Interface
    3. Leave its content empty
    4. Save your changes

    This step only needs to be done once.

    Step 3. Specify (or add) the item to appear as a card¶

    The last step is to create the cards. Each card is based on a single Blackboard content item.

    1. Create a ne Blackboard content item on your page.
    2. Give it the name you want to appear on the card.
    3. Add any content that you would like to appear on the card.
    4. Add to the content for this card, a line that starts with and contains only Card Image:
      It doesn’t have to be in bold, but it should include the colon.
    5. Save your work.

    Figure 1 shows a very simple Blackboard content item that has been prepared for being transformed into a card. Figure 2 shows what the final card will look like. You’ll note that it is missing an image. The “How to customise individual cards” section will show how you can customise cards by adding images and additional information.
